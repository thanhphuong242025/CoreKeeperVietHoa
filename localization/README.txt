This directory is installed with a file called "Localization Template.csv". DO NOT EDIT FILE.

Instead, make a copy of "Localization Template.csv" and rename it "Localization.csv".

The file can be opened in any Excel-like program as a .csv file, but using the tabulation character as a separator instead of a comma.

If you are only adding a single new language, then it is a good idea to remove columns with other languages from the "Localization.csv" file you're working on, to avoid them not being updated with newer translations.

Do keep a backup of any custom localization file since we, the developer, will automatically update the "Localization.csv" file as well with new entries.

To allow players to change existing translations as well without them being overwritten, existing entries will not be updated.

The "Localization Template.csv" file will have to be referenced to know which entries might have been updated, it'll need to be used as a point of comparison.
